ในช่วงสัปดาห์แรกผมเริ่มด้วยการใช้ AI ช่วยสร้างโจทย์เพื่อทบทวนความเข้าใจเกี่ยวกับ JavaScript โดยเน้นที่พื้นฐานและ Syntax 
เรื่อง Array, Object, OOP และ Function ซึ่งประกอบด้วยโจทย์หลายอย่าง เช่น การจำลอง fetch API เพื่อแสดงข้อมูลบนหน้าเว็บ 
การรวมข้อมูลจาก 2 Array ของ Object โดยใช้ find เหมือนการ join ใน SQL การสร้าง Class สำหรับจัดการตะกร้าสินค้า 
พร้อม methods สำหรับเพิ่มและลบข้อมูล รวมถึงการใช้ reduce เพื่อคำนวณราคารวม การนับคำซ้ำใน Array และแปลงเป็น Object 
การเขียนฟังก์ชันค้นหาสินค้าโดย case-insensitive การเปรียบเทียบ shallow equality ของ Object การสร้างและใช้งาน Class 
และ Function ต่างๆ จากนั้นในสัปดาห์ที่สองก็ได้ฝึกเขียนคำสั่งเกี่ยวกับการนับจำนวนตัวเลขซ้ำใน Array การกรอง Array ของ Object 
ตามหลายเงื่อนไข การแปลง Object เป็น JSON String การกลุ่มคำใน Array ของ String โดยใช้ reduce การสร้าง Class Student 
สำหรับคำนวณเกรด การใช้ export และ import รวมถึงเรียนรู้ async/await และ Promise ทั้งในแง่ของการรอผลลัพธ์ 
การใช้ Object.entries โดยนำข้อมูลมาแปลงและจัดกลุ่ม รวมถึงการใช้ spread operator ในการ merge และจัดการ Object ต่างๆ 
นอกจากนี้ยังได้เรียนรู้การเรียกใช้งาน destructuring สำหรับการจัดการกับ Object และ Array อย่างมีประสิทธิภาพ การเขียนฟังก์ชันต่างๆ 
เช่น การเพิ่ม Property, การ merge Object, การนับคำและตัวอักษรซ้ำ การแปลงข้อมูลเป็น Uppercase การหา Max Value 
รวมถึงการกรองค่าที่เป็น falsy และการดำเนินการคำนวณต่างๆ เช่น การคูณ การยกกำลัง การคัดกรองเลขคู่หรือคี่ เพื่อพัฒนาทักษะทั้งในด้าน logic 
การเขียนโค้ดแบบ Functional และ OOP ของผมไปในทิศทางที่ถูกต้องและการนำความรู้ไปใช้ในงานจริงได้มากขึ้น
