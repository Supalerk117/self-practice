Self learning ครั้งแรกผมอยากทวนความจำและเนื้อหาเทอมที่แล้วก่อนเลยเลือกที่จะให้ AI ช่วย generate โจทย์มาให้ทำโดยผม prompt ว่า
"ขอโจทย์ javascript ที่น่าจะได้ใช้จริงในตำแหน่ง frontend อยากได้เกี่ยวกับ array และ object OOP function สัก 5 ข้อ"
โดยคาดหวังว่าจะได้ทบทวนความรู้พื้นฐานและ syntax ของ javascript เรื่องการใช้ Array , Object , OOP และ Functinon

1.การจำลองว่าเรา fetch API มาจาก backend แล้วนำข้อมูลมาแสดงในหน้าเว็บ 
ข้อนี้ได้ฝึกการใช้ filter กับ map แบบพื้นฐานและได้ทวนความจำ syntax ของ Object

2.การรวมข้อมูลจาก 2 Array ของ Object ตาม id
ได้ฝึกการใช้ find เพื่อหาตัวเชื่อมระหว่าง Obj เหมือนการ join table ใน sql

3.สร้างคลาส OOP สำหรับจัดการตะกร้าสินค้า
ได้ทวนความจำการใช้ class , การสร้าง method ให้กับ class คือ addItem กับ removeItem โดย addItem ก็แค่ push เข้าไปใน Array
ส่วน removeItem ผมเลือกใช้วิธีการ filter ตัวมันเองที่ไม่ได้มีชื่อที่ต้องการลบ ใช้วิธีนี้เพราะมันง่ายและรวบรัดดี
และได้ทบทวนการใช้ reduce โดยการหาค่า total price ของทุก item

4.นับจำนวนคำซ้ำใน Array และแปลงเป็น Object
ข้อนี้คิดว่าค่อนข้างยาก ตอนแรกผมลองใช้ for loop แต่คิดต่อไม่ได้เลยลองให้ AI ช่วยไกด์ทางดูเลยรู้ว่าควรใช้ reduce โดยมีค่าเริ่มต้นเป็น empty Obj

5.ฟังก์ชันค้นหาสินค้า (Case-insensitive)
ข้อนี้ไม่ได้ลงมือเขียนเองเพราะไม่รู้ว่าควรเริ่มจากตรงไหนเลยให้ AI generate มาแทน ข้อนี้ได้เห็นวิธีการใช้ .filter() เพื่อเลือกสินค้าที่ชื่อมี keyword อยู่
โดยที่ไม่สนตัวพิมพ์เล็กหรือใหญ่ และแปลงทั้ง keyword และชื่อสินค้าเป็น lowercase ด้วย .toLowerCase() เพื่อเทียบแบบไม่ case-sensitive
